# C64_PrologicDos
Redraw of Prologic-Dos Classic pcb's for the Commodore 64

Prologic-Dos is a Commodore 64 parallel speedloader.
It contains a pcd for the Commodore 64 and a pcb for the 1541 diskdrive.
